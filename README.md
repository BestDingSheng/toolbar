# toolbar

---
[预览本站](http://box.jsding96.cn/)

## What’s toolbar

本站是基于 hexo + hexo-theme-webstack 主题构建的一个静态网站

## Why toolbar

主要把平时工作生活中一些常用的网站给收藏起来，进行归纳和整理方便自己查找

## Installation

```bash
git clone https://github.com/BestDingSheng/toolbar.git
```

## Usage

```bash
cd toolbar
npm install 
npm run dev
```
